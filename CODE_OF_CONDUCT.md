# Code of Conduct

## Our Standards

Fynor is a technical project. We expect all participants to:

- Be direct and specific. Vague criticism wastes everyone's time.
- Be respectful of other contributors' time and work.
- Argue about ideas, evidence, and code — not about people.
- Accept that disagreements about ADR amendments are expected and healthy.
  Use evidence and citations, not authority or seniority.
- Give credit where it is due.

We do not tolerate:

- Personal attacks, harassment, or discriminatory language.
- Deliberate misrepresentation of other contributors' positions.
- Sustained disruption of technical discussions.

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests,
code review comments, and any other venue where Fynor contributors interact.

## Enforcement

Violations should be reported to **sriram@fynor.tech**. All reports will be
reviewed promptly. The response may include a warning, temporary ban, or permanent
ban, depending on the severity and history of the violation.
